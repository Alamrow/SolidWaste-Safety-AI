User Guide

Getting Started

Launching the App: Open the application from your desktop. The main dashboard is designed for intuitive navigation, with clearly labeled sections for data import, AI predictions, results visualization, and export options.

Dashboard Overview: Your dashboard provides quick access to all major functions. Tooltips and brief descriptions accompany each icon, so you always know what each option does.

Importing Your Data

Prepare your dataset properly before importing:

  • city: Name of the city.
  • time: Date and time (formatted as YYYY-MM-DD HH:MM).
  • latitude & longitude: Geographic coordinates.
  • population: City population number.
  • climate_zone: Classification of the climate type.

Example CSV file structure :

To visualize and compare your AI predictions with actual data, you can include an optional column named user_defined in your CSV file. This column should contain the predicted values for the environmental parameter you are forecasting.

Example: Dubai,2020-12-01 12:00,25.276987,55.296249,3300000,Desert,24.5,180,120,4500,35,65,85,4500,120,1015,0.5,60,12.5,400,220,2.5,65,0.3,8.5,Your AI prediction Value

Generating Reports

1. Exporting Waste & Safety Report

To generate a comprehensive Waste & Safety Report for a specific location:

  1. Navigate to the "Location Info" section of the application.
  2. Ensure the desired location is currently displayed.
  3. Click the "Generate Full Report" button.

Upon clicking, a full Waste & Safety Report will be generated. You will then have the options to:

  • Print the report directly.
  • Save as PDF or other supported document formats.

2. Exporting Prediction Data Report

To generate a report of your prediction results for a chosen environmental parameter:

  1. First, run a prediction for your desired environmental parameter.
  2. Once the prediction is complete, click the "Export Prediction Data" button.

This action will generate a report containing the prediction data for the selected parameter.

3. Downloading Plots as PNG

You can easily save any generated plot or chart as a PNG image:

  • Locate the "DOWNLOAD Plot" button associated with the specific plot you wish to save.
  • Alternatively, look for an "Export Chart" option (which may offer additional formats).
  • Click the respective button to download the plot as a PNG file.

Configuring AI Predictions

Set up your prediction by adjusting the following options:

  • Prediction Model: Choose from:
    • Linear Regression
    • Random Forest
    • Polynomial Regression
    • ARIMA
    • Neural Network
  • Chart Type: Line, Area, Bar, Scatter, or 3D Surface.
  • Forecast Years: Select between 1-10 years.
  • Confidence Level: Adjust between 80%-99% for accuracy control.
