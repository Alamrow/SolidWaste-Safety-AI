CSV Import Guide

Your CSV file should include these columns:

city - City name
time - Time point (date/hour)
latitude, longitude - Coordinates
population - City population
climate_zone - Climate type

Plus any of these parameters:

Example: Abu Dhabi,2024-01-01,24.4667,54.3667,1580000,BWh,3600,45,50,1600,2000,70,4900,110,25,93,86,380,20,9,9
